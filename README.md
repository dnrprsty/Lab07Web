# Praktikum 7 – PHP Dasar

## Tujuan
Buat ngenalin dasar-dasar **PHP**, mulai dari cara kerja script **server-side**, bikin **variabel**, **kondisi**, **perulangan**, sampe bikin program PHP sederhana yang bisa jalan di browser lewat **Apache (XAMPP / LAMPP)**.

## Persiapan
1. Install **XAMPP / LAMPP** biar ada Apache + PHP-nya.  
2. Jalanin server:
   ```bash
   sudo /opt/lampp/lampp start
3. Buka browser → ketik http://localhost/ buat cek server aktif.
